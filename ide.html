IDE's sugeridos para la implementación de Apps

  • ¿Qué es un IDE?

    Un IDE, por otro lado es un entorno de desarrollo listo para realizar distintas tareas, tales como debutar código, accesos rápidos para encontrar las relaciones entre unos métodos u otros y en general dar al desarrollador una herramienta capaz de trabajar con frameworks y tirar adelante el proyecto.

  • Xcode

    Es un entorno de desarrollo integrado (IDE) creado por Apple para desarrollar aplicaciones (iOS, macOS, watchOS y tvOS). Proporciona un conjunto de herramientas para construir y probar aplicaciones. Una vez que se ha completado el desarrollo, Xcode permite empaquetar la aplicación y enviarla al App Store. Su editor de código permite diferentes lenguajes de programación como: C, C ++, Objective-C, Objective-C ++, Java, AppleScript, Python, Ruby, ResEdit y Swift. Xcode solo se ejecuta en plataformas macOS.

  • Android Studio

    • Android Studio él es IDE oficial para el desarrollo de aplicaciones Android, está basado en IntelliJ IDEA. Incorpora herramientas que ofrecen distintas funciones como: compilación flexible, entorno unificado para todos los dispositivos Android, integración con GitHub, identificador de problemas (rendimiento, usabilidad y compatibilidad) entre otras. Admite lenguajes de programación como Java y Kotlin. Android Studio está disponible para plataformas de escritorio como Mac, Windows y Linux.
